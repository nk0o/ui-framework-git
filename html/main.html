Grove UI Framework Getting Started

Getting Started

아름다운 웹 사이트를 손쉽게 구현하세요.
UI Framework는 프로젝트 진행 시 웹 화면의 전반적인 구성 요소들의 일관성을 유지하고 팀 내부의 유연한 업무를 돕는 컴포넌트 시스템입니다.

간단하게 시작하세요.

기본 패키지를 다운로드 받아 프로젝트를 생성하세요.
필요한 소스 코드를 복사하여 디자인에 맞게 테마를 변경하여 사용하세요.

File list

로컬 폴더에 "filelist.html" 파일을 소스 코드를 확인합니다.
작업 진행 상황에 따라 아래와 같이 작성하고, 진행률은 자동으로 계산됩니다.

<tr class="list">
      <td>MAIN</td>
      <td><a href="url" target="_blank">filename.html</a></td>    /* 파일 이름 및 링크 */
      <td>date(ex 2022.05.13)</td>    /* 파일 생성 및 최종 수정일 */
      <td><span class="r"></span></td>    /* 페이지 구분:반응형 웹, 데스크탑 웹, 모바일 */
      <td><span class="end"></span></td>    /* 진행 상태:진행중, 수정, 완료 */
      <td><p>empty</p></td>    /* 참고사항 */
</tr>

Web font

기본 폰트는 내장되어 있으며 국문은 노토산스, 영문은 팝핀스가 적용되어 있습니다.
프로젝트에 따라 다른 폰트로 손쉽게 변경할 수 있습니다.

//Default font .font_kr{font-family: $font-noto;}
.font_en{font-family: $font-pop;}
/* 기본 폰트는 국문: 노토산스, 영문:팝핀스 폰트가 적용되어 있습니다.*/

//Embeded
$font-aritadt    /* 아리따 돋움 */
$font-aritasans    /* 아리따 산스 */
$font-lgsmha    /* LG 스마트체*/
$font-APHQ    /* 아모레HQ */
$font-SUIT    /* SUIT */

Key Point

컴포넌트는 웹 사이트를 구성하는 공통 요소입니다.
프로젝트마다 컴포넌트를 새로 제작하는 비효율적인 프로세스에서 벗어나 공통으로 사용되는 요소를 도출하여 각 컴포넌트에 적용하면 재사용이 가능합니다.
그 결과 작업 시간은 단축되고 더욱 체계적으로 프로젝트를 수행할 수 있습니다.

  • Standards

    웹 가이드를 준수하여 소스 코드의
    일관성을 유지합니다.

  • Performance

    손쉬운 사용과 소스 코드 최적화로
    효율 높은 컴포넌트를 생성합니다.

  • High Quality

    디자인 심미성이 반영된 컴포넌트를
    제공하여 초기 품질을 보장합니다.

  • Updating

    더 좋은 컴포넌트를 제공하기 위한
    정기적인 업데이트를 진행합니다.

Supported browsers

우리는 웹 페이지의 상호 호환성(Cross Browsing)을 준수합니다.
표준 웹 기술을 도입하여 다른 기종이나 플랫폼에 따라 달리 구현되는 기술을 비슷하게 만들고 어느 한쪽에 치우치지 않도록 공통 요소를 사용하여 웹 페이지를 제작합니다.

  • Edege

  • Chrome

  • Safari

  • Firefox

  • Opera

  • Explorer

We don't support Internet Explore 9, 10
